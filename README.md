## ✨ Overview

**Flora** is a frontend web project designed with a floral aesthetic and minimal UI approach.
It focuses on creating a visually pleasing layout while maintaining clean structure, responsiveness, and performance using vanilla web technologies.

---

## 🎯 Features

<p align="center">

🌿 Clean and minimal UI design
📱 Fully responsive layout (mobile-first)
⚡ Lightweight & fast performance
🎨 Modern CSS styling and layout system
✨ Smooth transitions and UI interactions
🧩 Well-structured and maintainable code

## 🎨 UI Philosophy

This project follows a simple design approach:

* Nature-inspired visual theme 🌸
* Soft spacing & clean typography
* Balanced white space usage
* Subtle animations for better UX
* Mobile-first responsiveness
